About AutomatedDeploy:
----------------------
This service lets you run a batch file at specific intervals of time, based on this condition:
  "On change of file size of a configurable file" OR "on change of last modified date of a configurable file"
The configuration can be done in the settings.config file provided with the Service Executable.

Uses:
-----
1. Can be used for automated deployment of a web application to IIS.
2. Can be used for automated deployment of a windows service.

Steps to Install:
-----------------
Use the setup.bat file to install the service.
